## üß™ TESTING TIPS

1. **Test Real Examples First**: Start with the real job postings to verify the model correctly identifies legitimate jobs.

2. **Test Fake Examples**: Use the fake examples to see how the model detects fraudulent postings.

3. **Mix and Match**: Try combining elements from real and fake postings to test edge cases.

4. **Partial Information**: Test with incomplete information (e.g., only title and description, no requirements).

5. **Common Red Flags to Test**:
   - Requests for upfront payment
   - Promises of unrealistic earnings
   - No interview process
   - Requests for personal/financial information
   - Urgency and pressure tactics
   - Poor grammar and spelling
   - Vague job descriptions
